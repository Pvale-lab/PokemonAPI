# PokemonAPI
Baixe os arquivos:

        Certifique-se de que os três arquivos (index.html, style.css, script.js) estejam na mesma pasta.

    Abra no Navegador:

        Navegue até a pasta onde salvou os arquivos.

        Dê um duplo-clique no arquivo index.html.

        O arquivo será aberto automaticamente no seu navegador padrão.

Como Usar a Aplicação

    No campo de busca, digite o nome (ex: "pikachu", Charizard) ou o ID (ex: "25", "132") do Pokémon que deseja encontrar.

    Clique no botão "Buscar".

    Se o Pokémon for encontrado, um card aparecerá abaixo com seu Nome, ID e Tipo principal.

    Se o Pokémon não for encontrado, uma mensagem de erro será exibida.
